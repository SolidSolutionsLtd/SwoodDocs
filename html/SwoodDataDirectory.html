Overview

The Swood Data Directory contains the libraries (Materials, Frames, SwoodBoxes, Connectors, etc.) as well as the report files used by Swood.

Note that this location is custom and can be defined locally or can be pointed to a mapped server directory.

How to find your Swood Data Directory

You can find the location of the Swood Data Directory by opening SolidWorks, then use the top menu to search for Tools > SWOOD Design > Settings

Please make sure to restart SolidWorks if the Swood Data Directory is changed.
