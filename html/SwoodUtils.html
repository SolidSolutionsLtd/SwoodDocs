Overview

Swood Utils streamlines the ID assignment process for panels, providing a user-friendly interface. It also identifies identical panels, removing the need for the “Combine Panels” macro.

Features

  • Assign IDs to panels
  • Remove IDs from panels
  • Identify identical panels and assign the same ID
  • Modify custom properties related to the Swood Report (e.g., Frame, Hardware, Exclude)

Types of IDs

Swood Utils supports two ID schemas: Flat and Nested.

Flat IDs

Flat IDs assign sequential IDs to panels.

Nested IDs

Nested IDs also assign sequential IDs, but include a reference to the parent frame.

Controls

  1. Refresh Data
  2. Assign TempIDs
  3. Save TempIDs to Models
  4. Remove TempIDs from Models
  5. Update View Orientation

Refresh Data

Refreshes data from SolidWorks, for example, after adding new components to an assembly or changing the active model.

Assign TempIDs

Assigns Temporary IDs to models. This process automatically runs after using Refresh Data.

Save TempIDs to Models

Saves Temporary IDs as custom properties in the model.

Remove TempIDs from Models

Removes ID custom properties from models.

Update View Orientation

This option reorients all active models to align with the Front view normal to Z-axis. Use this only when "CombineIdentical" is enabled and if a warning appears on one or more panels.

Workflow

  1. Open the assembly containing the panels to which you want to add IDs.
  2. Click the Refresh Data button to load data into Swood Utils.
  3. Once loaded, Swood Utils automatically assigns TempIDs to each panel based on your settings. These TempIDs serve as a preview and are not yet saved to the panel.
  4. After reviewing the TempIDs, click Save TempIDs to save them as custom properties for all panels.

ID Settings

Nested IDs

Select between Nested or Flat IDs.

Persistent IDs

Available for Flat IDs only. Keeps the current panel ID instead of overwriting it.

Reuse IDs

Available for Flat IDs only. Reuses unused IDs.

Combine Identical

Combines identical panels (based on geometry, material, edgebands, laminates, and grain direction) and assigns the same ID to all.

Order Type

Specifies how IDs are ordered:

  • None: Random order (fastest method)
  • SortSettings: Order IDs based on model custom properties, such as Thickness or Material
  • FeatureTreeOrder: Follows the feature tree order (slower method)

ID Separator

Character(s) used to separate IDs.

Frame Alpha ID

Uses letters for Frame IDs.

Frame Starting ID

Specifies the starting value for Frame IDs.

Frame Increment Value

Value used to increment Frame IDs.

Frame ID Format

Defines the format for numeric Frame IDs.

Frame Fixed Prefix

Fixed prefix added to all Frame IDs.

Frame Sort Order

Specifies the order used to sort the feature tree.

Panel Starting ID

Specifies the starting value for Panel IDs.

Panel Increment Value

Value used to increment Panel IDs.

Panel ID Format

Defines the format for numeric Panel IDs.

Panel Fixed Prefix

Fixed prefix added to all Panel IDs.

Panel Sort Order

Specifies the order used to sort the panel list.

Top-Level Property Prefix

Prefix for top-level properties.

Frame Level Property Prefix

Prefix for frame-level properties.
