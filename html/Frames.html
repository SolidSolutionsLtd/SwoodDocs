Overview

The Report Pro allows users to classify Assemblies or Multiple Stock Parts as a Frame. The Report Pro will list all Frames in an assembly and give detailed information of the panels, hardware and programs within the Frame.

The Frames tab will appear if there are 1 or more Frames in a project.

How to define a Frame

To define a Frame, add the custom property "Frame" and set the value to "Yes"
