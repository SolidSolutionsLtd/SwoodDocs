Overview

The report allows users to generate Cutting Patterns for panels of a specific Materials. This feature is aimed for Beam Saw optimisation, but can also be used to estimate the number of boards required for a project.

Requirements

  • Swood Design
  • Swood 2024 SP0.0
  • Report v2.10.0

How to generate a Cutting Pattern

To generate a Cutting Pattern requires defining Board sizes for the different materials and enabling For Board Estimation
