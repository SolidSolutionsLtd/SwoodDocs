Overview

Materials can be grouped using the Material's Extended Custom Property Material Type. This grouping provides a clearer breakdown of your panels in the report and summary.

Creating the Material Type property

To create the Material Type property, go to the Extended Properties in the Swood Materials Library. Create a new property by filling in the blank row in the Name column. The property should be a String Type with a default value of Core.

Note that adding a Material Extended Property requires restarting SolidWorks.

How to Define a Material Type

To define the Material Type of a specific material, you need to edit the material in your Material Library. Scroll down to the Extended Properties menu, tick the checkbox next to Material Type and define your Material Type name in the input field.

To define the Material Type for a specific material, edit the material in your Swood Materials Library. Scroll down to the Extended Properties menu, check the box next to Material Type, and enter your Material Type name in the input field.

Note that Material Type names are case-sensitive

Grouping by Material Type in the Report

When two or more Material Types are present in the report, a toggle option will appear in the top right corner of the Panel and Stock. Using this toggle will group the panels by their corresponding Material Types, as shown below.

Material Types also controls grouping on the Summary, displaying the total costs for the different Material Types.
