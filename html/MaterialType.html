Overview

Material can be grouped by using the Material's Extended Custom Property Material Type. Grouping materials in this way allows a clearer breakdown of your panels in the report and summary.

Creating the Material Type property

The Material Type property can be created in Extended Properties, which is found in the Materials Library. You can create the new property by filling in the blank row in the name column. The property needs to be a String Type and the default value should be Core.

Note that adding a Material Extended Property will require a SolidWorks restart.

How to Define a Material Type

To define the Material Type of a specific material, you need to edit the material in your Material Library. Scroll down to the Extended Properties menu, tick the checkbox next to Material Type and define your Material Type name in the input field.

The group names are case sensitive, so if you want it to join an existing Material Type, it needs to have the same spelling and capitalisation.

Grouping by Material Type in the Report

If two or more Material Types are present in the report, you will notice a toggle option in the top right corner of the Panel and Stock page. Using the toggle will group the panels into their corresponding Material Types as shown below.

The Material Type also controls the grouping on the Summary page, showing you total costs of the different Material Types.
