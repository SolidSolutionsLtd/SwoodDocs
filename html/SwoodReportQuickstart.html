Solid Solutions License Manager

All versions of the Swood Report are licensed. To activate the license you will need to download and install the Solid Solutions License Manager by clicking here.

Once this is done, please activate the provided license by launching the Solid Solutions License Manager and pressing the Add License button in the image below. Learn More.

Swood Editor App

To set up the Swood Report please start by downloading and installing the Swood Editor app by clicking here. If download doesn't start, please copy a paste the following link on a new tab http://downloads.solidsolutions.co.uk/swood/SwoodEditor-Setup-1.0.0.exe

The Swood Editor app allows you to modify report settings like columns, sorting orders, output location, as well as change default Swood settings. This app is currently in the Beta stage and we would appreciate any feedback for improvements.

Please note that the Swood Editor app uses the same license as the Swood Report, therefore ensure that your license is active before launching.

Setup the Swood Report

All of the report settings are stored withing the DAT folder which is contained within the Swood Data Directory. Learn More.

Launch the Swood Editor app and check if the DAT folder is the one within your Swood Data Directory.

The app will automatically search for the DAT, however you can also manually define it by clicking or dragging and dropping the folder in the specified field.

You can then click the Reset Report option to install a fresh instance of the report, or select Update Report to keep your previous settings. Please note that in both cases the app will always create a backup of your current DAT folder.

Please make sure to restart SolidWorks for the changes to take effect.
