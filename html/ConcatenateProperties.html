Overview

Properties can be concatenated or evaluated when showing them on a table. This allows users to create custom properties to be displayed on a table

How to Concatenate Properties

To concatenate properties, define a new property in the SwoodEditor and wrap the properties to be concatenated between chevron characters, for example Panel Dimensions: «Length» x «Width». In this example, for a 1000x500 panel, the table will display Panel Dimensions: 1000 x 500.

To add the opening chevron character use Alt+174.

To add the closing chevron character use Alt+175.

How to Evaluate Properties

To evaluate properties, define a new property in the SwoodEditor and wrap the properties with «EVAL("PropertiesToEvaluate")», for example Panel Area: «EVAL(«Length»*«Width»)». In this example, for a 1000x500 panel, the table will display Panel Area: 500000.

How to Concatenate and Evaluate Properties

Users can use a combination of concatenate and evaluate, for example Panel Dimensions: «Length» x «Width», Panel Area: «EVAL(«Length»*«Width»)». In this example, for a 1000x500 panel, the table will display Panel Dimensions: 1000 x 500, Panel Area: 500000.
