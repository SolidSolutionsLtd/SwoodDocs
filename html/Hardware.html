Overview

The report allows users to classify Parts or Assemblies as Hardware.

The Hardware tab will appear if there are one or more hardware components in a project.

How to Define Hardware

A component will be classified as Hardware if its filepath contains /Hardwares/. This means that a component will be defined as Hardware if it is saved inside a folder (or any of its subfolders) called Hardwares. Note that by default, Swood saves all hardware components in ".../Swood Data Directory/SWOODDesign/Hardwares/".

The Hardware classification will also be assigned if a component's filepath contains \SOLIDWORKS DATA. This ensures that all Toolbox components are classified as Hardware.

To manually define a Hardware component, add the custom property Hardware and set the value to Yes.
