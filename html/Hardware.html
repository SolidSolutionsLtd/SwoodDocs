Overview

The Report allows users to classify Parts or Assemblies as Hardware.

The Hardware tab will appear if there are 1 or more Hardware components in a project.

How to define a Hardware

The Hardware classification will be assign to Parts and Assemblies if their Filepath contains "/Hardwares/". This means that a component will be defined as Hardware if it is saved inside a folder (or any of its subfolders) called "Hardwares". Note that by default Swood saves all Hardware components in "Swood Data Directory/SWOODDesign/Hardwares"

The Hardware classification will all be assign is component's Filepath contains "\SOLIDWORKS DATA". This is to ensure that all Toolbox components are classified as Hardware.

To manualy define a Hardware, add the custom property "Hardware" and set the value to "Yes".
