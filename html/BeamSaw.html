Overview

Panels can be tagged as Beam Saw panels to distinguish which ones need to be sent for cutting on a Beam Saw. Tagging a panel will add a saw icon next to the parts on the Panels page, and create a new dropdown option in the Stocks tab dedicated to Beam Saw parts.

Another benefit of identifying Beam Saw panels is that the report will generate a separate set of labels for them.

Note that Beam Saw panels will not be displayed on the Stocks page.

How to Define a Beam Saw Panel

To define a Beam Saw Panel, add the custom property Beam Saw and set the value to Yes.
