Overview

Panels can be tagged as Beam Saw panels to distinguish which need to be sent to be cut on a Beam Saw. Tagging the panel will add a saw icon next to the parts on the Panels Page, as well as creating a new dropdown option in the Stocks Tab, dedicated to the Beam Saw parts.

Another benefit of identifying Beam Saw panels is that the report will generate a separate set of labels for them.

Note that Beam Saw panels will not be showed in the Stocks page.

How to define a Beam Saw Panel

Add the custom property Beam Saw and set the value to Yes.
