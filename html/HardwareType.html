Overview

The report enables users to classify hardware components by Type, providing a clearer breakdown by grouping different types into separate tables.

How to define a Hardware Type

To define a Hardware Type, add the custom property Hardware Type and set the value, such as, Handle, Connector, Hinge, Miscellaneous, etc.

When Hardware Types are present in a project, you can use the Type toggle to group the different hardware types into separate tables, as shown below.
