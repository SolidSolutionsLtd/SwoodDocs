Issues with long file names

  • Overview
  • Ways to approach the problem
    • Rename or move DAT folder
    • Define a simplified naming convention
    • Swood Automatic Numbering System

Overview

Microsoft Windows has a maximum path limit of 260 character. If the length of the path and filename combined reach this limit, Windows will not be able to save the file to the directory.

The way Swood creates filenames is by concatenating the names of parts and assemblies. Whilst this generates a unique filename, it also tends to create a long filename.

There are a few way to approach this:

  • Rename or move DAT folder
  • Define a simplified naming convention
  • User the automatic numbering system from Swood

Rename or move DAT folder

Rename or move the Swood Data Directory folder closer to the root folder of your PC to avoid unnecessary nested directories.

Define a simplified naming convention

Create a simplified naming convention for Parts and Assemblies, something compromised of a couple of letters and numbers.

As a simple example, “HC1_DR1_T” instead of “High Cabinet Type 1_Drawer Runner Type 1_Top”

A simplified naming conventions will not allow for a descriptive name. However, you can add a Description custom property to the file and show it in the Swood Task Pane. You will need to enable this setting in Tools > SWOOD Design > Settings > Design > Frames > Show descriptions

Swood Automatic Numbering System

Limitations

  • The Swood Data Directory cannot be shared in Dropbox or any similar file sharing service due to latency.
  • Used counters cannot be retrieved/reused. Once they are used, it cannot be undone.
  • The counters carry on from the previous project, it cannot be reset to 1 for a new project.

Overview

The Swood Automatic Numbering System automatically assigns names/numbers to components. The tool lets you define prefixes to 3 categories, Assemblies, Sub-Assemblies and Parts; in the example below we have “Frame”, “SubFrame” and “Panel” respectively. Each category has its own counter that increments each time a component is created.

You can assign counters to all components when using the "Insert Copy" option for Frames and SwoodBoxes.

Selecting the User Serial Number option will rename each component using the predefined prefix and the next serial number in the sequence. Notice that this method ensures that each component has a unique name, so there is no longer the need to append the assembly name.

To assign a counter to a panel when using the Create a new panel option, use the Use EPDM SerialNumber tick option.
