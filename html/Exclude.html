Exclude Components from Report

  • Overview
  • How to define a Hardware

Overview

Users are able to exclude components from the report.

When one or more components are excluded, the Panels tab will display a yellow dot next to it.

You are also able to see all excluded components by selecting the Excluded Items tab within the Panels tab.

How to Exclude a Component

To exclude a component from the report, add the custom property Exclude and set the value to Yes
